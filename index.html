Sass

Einführung

Sass ist eine Erweiterung für CSS. Sie bietet viele Funktionen und hat eine sehr ähnliche Syntax wie CSS selbst. Es soll CSS nicht umändern, sondern nur das korrigieren, was bei CSS fehlt. Es ist eine gute Hilfe zum Beibehalten der Übersichtlichkeit. Es werden auch Variablen mit Sass mitgebracht, welche das wiederholte Benutzen von Werten vereinfacht.
